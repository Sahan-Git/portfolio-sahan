Laravel

Pour ce second projet de groupe, nous devions créer un site pour les fiches des visiteurs d'une industrie pharmaceutique mais cette fois avec l'aide de Laravel.

Cette application web est destinée aux visiteurs médicaux et personnels du service comptable, les premiers pour renseigner et consulter leurs états de frais, les seconds pour réaliser le suivi des états de frais des visiteurs médicaux jusqu'à leur règlement.

Le site doit être adapté pour les supports mobiles.
Elle utilise le framework Laravel, l'ORM Eloquent et une base de données MySQL locale.
